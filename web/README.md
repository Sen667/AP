# Mission 3 : Obligation d'inscription au Ram « Les Fripouilles »

## Contexte

Étant donné que les Relais d'Assistantes Maternelles relèvent d'une communauté de communes, tous les services proposés doivent tendre à être proposés aux personnes concernées. Afin de diligenter les renseignements à des personnes appartenant au territoire concerné, toute personne souhaitant se rapprocher du Ram doit s'inscrire pour bénéficier de ses services (information, application, garde).

Le compte ainsi créé sera utilisé pour toutes les étapes d'authentification sollicitées par les différentes applications.

## Travail à Faire : (application légère) (site Web)

Afin d'identifier l'ensemble de son public, le Ram souhaite disposer d'une application type web destinée à l'inscription de toutes les personnes intéressées ou concernées par ses activités.

Cette application est principalement destinée à l'inscription des parents et des assistantes maternelles.

### Exigences fonctionnelles :

1. **Collecte d'informations** : Pour chacun des profils (parents et assistantes maternelles), il faudra :
   - Des informations nominatives
   - Des informations relatives aux enfants à prendre en charge (dans le cadre d'ateliers futurs par exemple)

2. **Structure des données** : Il est essentiel que la structure des informations enregistrées dans la base de données permette une exploitation dans les diverses autres applications développées.

3. **Fonctionnalités** : L'application devra permettre :
   - De s'enregistrer (création de compte)
   - De modifier les données de son compte ainsi créé

4. **Conformité légale** : Vous veillerez à mettre en place les mentions légales à indiquer dans le cadre de l'usage de données à caractère nominatif.

5. **Accessibilité** : L'application sera exploitable sur smartphone.
